# The Volumn TDD Workflow

## Default Sequence

1. Identify the owning folder and affected packages.
2. Read `PRD_V2.md`, this folder, and the relevant local `AGENTS.md`.
3. Identify the exact tests to add or update first.
4. Create or update those tests.
5. Implement the smallest change needed to satisfy them.
6. Sync contracts, docs, and local guides.
7. Run the relevant tests.
