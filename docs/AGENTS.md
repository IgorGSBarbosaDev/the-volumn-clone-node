# Docs Folder Agent Guide

## Purpose

This folder owns durable project documentation.

## Owns

- product documentation,
- API documentation,
- architecture notes,
- ADRs,
- AI operating docs.

## Rules

- Keep docs aligned with `PRD_V2.md`.
- Product decisions belong in the PRD.
- hard-to-reverse technical decisions belong in `docs/adr`.
- implementation guidance belongs in `docs/architecture`.
- public contract docs belong in `docs/api`.
